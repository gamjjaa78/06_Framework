요청위임된 request에 존재하는 파라미터 출력하기

    ${param.key}
    -request 존재하는 parameter 값 얻어와 출력
    -요청url에 쿼리스트링(GET)이나 폼데이터(POST)로 전달된 파라미터 값을
     직접 req에 세팅안해도 바로 접근할 수 있게 해줌
  • 제출된 이름
  • 제출된 나이
  • 체크된 색상

th:text/th"utext

    th:text="속성값"
    -해당 태그에 "속성값"을 내용으로 출력
    -단, html 태그, 특수문자 해석x (innerText)

    th:utext="속성값"
    -해당 태그에 "속성값"을 내용으로 출력
    -단, html 태그, 특수문자 해석ㅇ (innerHTML)
